# Research Loops

This folder stores online-research notes generated during the active
critique/research/revise prompt workflow.

## Purpose

Each research note should capture:

- the prompt weakness being investigated
- the concrete research questions asked
- the official or primary sources reviewed
- the local prompting/BDA references re-checked afterward
- the prompt implications carried into the next version

## Folder Convention

Use a lab-first, version-first structure:

```text
Research_Loops/<lab>/<version>/<run_folder>/research.md
```

Example:

```text
Research_Loops/2.main__qwen3-vl_8b-instruct/v004/run01_2026-04-12_155635_EDT/research.md
```

## Working Rule

These notes complement the prompt-lab run artifacts. The run folder in the
prompt lab should record what happened; the paired research note should record
what we learned from outside references and how that should change the next
draft.
